# Laravel

## Deploy

```
cd laravel
npm install
composer install --ignore-platform-reqs
```

## Migrations

```
mv .env.example .env
php artisan migrate
```

## Localhost

```
php artisan key:generate
php artisan serve --host=localhost --port=4000
```

# React

## Deploy

```
cd react
yarn install
```

## Sass

```
sass --watch sass:src/assets/css
```

## Localhost

```
yarn start
```
